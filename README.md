## 1. 大規模カンファレンス（セッション・講演型）

複数の講演（セッション）やパネルディスカッションが並行して進行する大型イベントです。
業界の最新トレンドや企業の技術導入事例などをインプットするのに適しています。

### 1. プログラミング言語・フレームワーク

特定の開発言語やツールのコミュニティが主催する、最もエンジニア向けのカテゴリーです。

### 2. UI/UX・プロダクトデザイン

エンジニアとデザイナーの境界にある、設計やユーザー体験に特化したカテゴリーです。

### 3. クラウド・インフラ・セキュリティ

システムの基盤や運用、セキュリティに特化したカテゴリーです。
大手クラウドベンダーが主催する巨大イベントが目立ちます。

### 4. AI・データサイエンス

人工知能、機械学習、データ分析の最新トレンドや研究成果を扱う急成長中のカテゴリーです。

### 5. 開発プロセス・マネジメント（アジャイルなど）

技術そのものよりも、チーム開発の手法や組織づくりに焦点を当てたカテゴリーです。

## 2. ハンズオン（体験型・ワークショップ）

講演を聴くだけでなく、参加者自身がPCを持参して実際にコードを書いたり、ツールを操作したりする体験型イベントです。新しい言語やフレームワークを実践的に学ぶ際に活用されます。

## 3. ミートアップ（交流・ライトニングトーク型）

特定の技術やテーマに関心がある人が少人数〜数十人規模で集まるイベントです。参加者同士のネットワーキング（人脈作り）や、短時間で発表するライトニングトーク（LT）などが中心となります。

## 4. ハッカソン / アイデアソン（開発・競争型）

数日から1週間程度の短期間でチームを組み、特定のテーマに沿ったアプリケーションやサービスの開発（ハッカソン）、または新しいアイデアの創出（アイデアソン）を競い合うイベントです。
